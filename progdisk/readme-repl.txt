SwiftII - Help
==============

Version @VERSION@
Built @BUILT@

A Swift-like language for the Apple
][+ and //e. Ships as a 9-disk
set: 4 REPL disks, 4 compiler
disks, and the data disk.


MENU OPTIONS
  1 REPL - interactive prompt, on
    this disk's interpreter.
  2 File selector - browse files +
    .swift programs.
  3 Debug - hardware diagnostic
    (disk space, detection, memory,
    slots; arrows page, Esc exits).
  4 About - version + the disk set.


FILE SELECTOR KEYS
  Ret  open in the editor (or launch
       a .system file)
  X    execute / run a .swift
  E    edit
  F    new file
  R    rename
  D    delete
  N    new folder
  ,    re-pick the drive (drive 2 =
       the data disk + tests)
  W    toggle 40/80 col (//e only)

EDITOR KEYS
  arrows  cursor left/right
  Ctrl-O/L  cursor up/down
  Ctrl-D  backspace
  Ctrl-S  save
  Ctrl-R  save + run
  Ctrl-Q  back to the browser
  Ctrl-W  _ in Swift mode on ][+
  (open files from the browser)

Demos ship in SAMPLES/ on this disk.


THE REPL DISKS
  ][+ lite  SWIFTIIP any ][ or ][+
  ][+ sat   SWIFTSAT +Saturn 128K
  //e lite  SWIFTIIE any //e
  //e aux   SWIFTAUX +64K 80-col aux

  lite   - core language: int,
           string, array, funcs,
           if/while/for, optionals.
  extras - lite plus gr/plot,
           peek/poke, htab/vtab,
           asc/chr, array methods,
           80-col.
